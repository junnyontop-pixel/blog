## 🚧 현재 상태 (Beta)

## ✨ 주요 기능

- 📄 게시글 목록 / 상세 보기
- ✏️ 게시글 작성 / 수정 / 삭제
- 🧾 Markdown 지원
- 💡 코드 블록 하이라이팅
- 🔄 새로고침 시 데이터 유지 (Supabase 연동)
